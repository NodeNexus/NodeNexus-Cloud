# VNAV Cloud Development Rules

Frontend
- React
- TypeScript
- TailwindCSS v4
- Framer Motion
- Zustand
- React Query

Backend
- FastAPI
- Python 3.13
- Async APIs
- Pydantic v2
- Docker SDK

Architecture
- Component based
- Modular
- No duplicated code
- Strict TypeScript
- Type hints everywhere

UI
- Apple
- Linear
- Vercel
- Dark Theme
- Glassmorphism

Rules
- Never use inline CSS.
- Never use shell commands if Docker SDK exists.
- Everything modular.
- Every page reusable.
